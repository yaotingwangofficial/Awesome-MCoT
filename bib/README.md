# Bibliography Snapshot

`refs.bib` is a snapshot of the BibTeX file used by the companion survey paper.
It is the authoritative source for paper metadata (title, authors, venue, year)
referenced from the survey, and can be used to:

- Auto-generate citations in the README
- Build figures from a single source of truth
- Verify titles when adding new entries

## Sync

This file is **manually synced** from the paper's `refs.bib`. When the paper
adds new citations, copy the updated file here.

```bash
# from the paper repo:
cp path/to/paper/refs.bib bib/refs.bib
```
